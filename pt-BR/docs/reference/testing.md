# Opções de Teste

BMad fornece um agente QA integrado para automação rápida de testes e um módulo Test Architect (TEA) separado para testes avançados.

## QA Integrado (Quinn)

Use o agente QA integrado para cobertura de teste rápida e direta:

- Gatilho: `QA` ou `bmad-bmm-qa-automate`
- Melhor para: projetos pequenos, cobertura rápida, padrões padrão

## Módulo Test Architect (TEA)

TEA é um módulo standalone com workflows de teste avançados (design de teste, ATDD, automação, revisão, rastreamento, avaliação NFR).

- Documentação: <https://bmad-code-org.github.io/bmad-method-test-architecture-enterprise/>
- Instalar: `npx bmad-method@alpha install` e selecione o módulo TEA
