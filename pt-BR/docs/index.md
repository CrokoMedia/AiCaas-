O Método BMad (**B**reakthrough **M**ethod of **A**gile AI **D**riven Development - Método Revolucionário de Desenvolvimento Ágil Orientado por IA) é um framework de desenvolvimento orientado por IA que ajuda você a construir software através de todo o processo, desde a ideação e planejamento até a implementação agêntica. Ele fornece agentes de IA especializados, workflows guiados e planejamento inteligente que se adapta à complexidade do seu projeto, seja você corrigindo um bug ou construindo uma plataforma empresarial.

Se você está confortável trabalhando com assistentes de codificação IA como Claude, Cursor ou GitHub Copilot, está pronto para começar.

## O Que Você Vai Precisar

BMad funciona com qualquer assistente de codificação IA que suporte prompts de sistema personalizados ou contexto de projeto. Opções populares incluem:

- **[Claude Code](https://code.claude.com)** — Ferramenta CLI da Anthropic (recomendado)
- **[Cursor](https://cursor.sh)** — Editor de código com IA em primeiro lugar
- **[Windsurf](https://codeium.com/windsurf)** — IDE de IA da Codeium
- **[Roo Code](https://roocode.com)** — Extensão do VS Code

Você deve estar confortável com conceitos básicos de desenvolvimento de software como controle de versão, estrutura de projeto e workflows ágeis. Nenhuma experiência prévia com sistemas de agentes estilo BMad é necessária—é para isso que estes docs servem.
