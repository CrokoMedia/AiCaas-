Quick Flow é para quando você não precisa do Método BMad completo. Pule Product Brief, PRD e Arquitetura - vá direto para implementação.

## Como Funciona

1. **Execute `quick-spec`** — gera uma especificação técnica focada
2. **Execute `quick-dev`** — implementa

É isso.

## Quando Usar

- Correções de bugs
- Refatoração
- Recursos pequenos
- Prototipagem

## Quando Usar Método BMad Completo

- Novos produtos
- Recursos importantes
- Múltiplas equipes envolvidas
- Alinhamento de stakeholders necessário
