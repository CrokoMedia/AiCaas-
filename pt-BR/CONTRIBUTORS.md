# Contribuidores

O BMad Core, BMad Method e os Módulos Comunitários BMad são possíveis graças às contribuições de nossa comunidade. Agradecemos com gratidão a todos que ajudaram a melhorar este projeto.

## Como Creditamos Contribuidores

- **Histórico do Git** — Cada contribuição é preservada no histórico de commits do projeto
- **Badge de contribuidores** — Veja a lista dinâmica de contribuidores no nosso [README](README.md)
- **Gráfico de contribuidores do GitHub** — Representação visual em <https://github.com/bmad-code-org/BMAD-METHOD/graphs/contributors>

## Tornando-se um Contribuidor

Qualquer pessoa que envie um pull request que seja mesclado se torna um contribuidor. As contribuições incluem:

- Correções de bugs
- Novos recursos ou workflows
- Melhorias na documentação
- Relatórios de bugs e triagem de issues
- Revisões de código
- Ajudar outros nas discussões

Não há requisitos mínimos de contribuição — seja uma correção de erro de digitação de um caractere ou um recurso importante, valorizamos todas as contribuições.

## Direitos Autorais

O projeto Método BMad é propriedade intelectual da BMad Code, LLC. Contribuições individuais são licenciadas sob a mesma Licença MIT do projeto. Contribuidores mantêm crédito de autoria através do histórico do Git e do gráfico de contribuidores.
